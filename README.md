## 🚀 Running Locally
1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/A-b-h-i-n-a-V-01/github-portfolio.git
    cd github-portfolio
    ```
2.  **Launch the Portfolio:**
    Simply open the `index.html` file in your preferred web browser, or run a local lightweight server:
    ```bash
    # Python 3
    python -m http.server 8000
    ```
    Then visit `http://localhost:8000` in your browser.
